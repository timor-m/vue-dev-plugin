## vue-dev-overlay

> used at vue-dev-plugin



> #### if you want to develop the package , you must first configure the vue-dev-shared/config/index.js ,set IS_DEV = true
#### Usage

~~~
<div id="vue-dev-plugin-overlay"></div>
~~~



#### Support

~~~~
module.exports = {
    style: 'style code',
    script: 'script code'
}
~~~~



#### Develop

~~~
npm run dev
~~~
